# 问卷调查管理系统

#### Description
问卷调查管理系统拥有两种角色，分别为管理员和用户

管理员：用户的增加和删除、查看问卷结果、管理所有问卷、查看个人发布问卷

用户：查看所有问卷，填写问卷，创建问卷
